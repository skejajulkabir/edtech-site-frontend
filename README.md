### Run Locally

To run **EduTech** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/skejajulkabir/edtech-site-frontend.git
```

Windows:

```bash
git clone https://github.com/skejajulkabir/edtech-site-frontend.git
```
